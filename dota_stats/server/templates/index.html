HuskarMetrics: Dota 2 Analytics

Yet Another Dota 2 Analytics Site (YAD2AS), designed and built by a low MMR hack

Welcome!

Welcome to the opence-source Dota 2 data analytics hobby project, maintained by Dr. Jubbs and hosted on Github.

This site uses the Dota 2 web API to harvest match information to generate statistics and analytical insights on the current Dota 2 meta. Currently only a pareto-style analysis of win rates and overall service health metrics are being generated, with more content planned over the coming months.

The source code is licensed under the permissive MIT License. The site uses the Python Flask framework for the webserver, Plotly for charting, and MongoDB as the back-end database. sNext up enhancements include win rates by inferred position, and individual hero match-ups.
